## 권장 실행 순서

```bash
./scripts/docker/nwm-run.sh "
python scripts/text/generate_qwen_captions.py \
  --input-root /path/to/frames_1fps \
  --output /workspace/nwm/datasets/derived/text_pipeline/captions_raw.jsonl \
  --model-name-or-path /workspace/nwm/weights/pretrained/Qwen2-VL-7B-Instruct \
  --prompt-mode scene_only \
  --batch-size 4 \
  --dtype bfloat16
"
```

```bash
./scripts/docker/nwm-run.sh "
python scripts/text/clean_captions.py \
  --input /workspace/nwm/datasets/derived/text_pipeline/captions_raw.jsonl \
  --output /workspace/nwm/datasets/derived/text_pipeline/captions_clean.jsonl \
  --format sentence
"
```

```bash
./scripts/docker/nwm-run.sh "
python scripts/text/precompute_text_embeddings.py \
  --input /workspace/nwm/datasets/derived/text_pipeline/captions_clean.jsonl \
  --output-root /workspace/nwm/datasets/derived/text_pipeline/text_embeds \
  --model-name-or-path openai/clip-vit-base-patch32 \
  --batch-size 128 \
  --max-length 77 \
  --dtype float16
"
```

## 학습 config 메모

- `text_conditioning` 예시:

```yaml
text_conditioning:
  enabled: True
  embedding_root: /workspace/nwm/datasets/derived/text_pipeline/text_embeds
  condition_source: current
  text_dim: 0
```

- `text_dim: 0`이면 cache에서 자동 추론하도록 구현됨.

## 리스크 / 주의점

- `Qwen2-VL-7B-Instruct`는 실제 full dataset run 시 속도와 VRAM을 확인해야 함.
- frame time과 text cache time 정렬이 깨지면 dataloader에서 zero vector fallback이 들어갈 수 있음.
- 현재는 "구현 완료"와 "실행 검증 완료"를 구분해서 봐야 함.
- 일부 config 파일은 권한 문제로 직접 수정이 안 된 것이 있어, 필요 시 쓰기 가능한 새 config를 만드는 편이 나음.
